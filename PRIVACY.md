# 声页隐私说明

声页以“本地优先”为设计原则。

## 不收集的数据

- 不上传或分析用户导入的 EPUB、TXT 内容
- 不上传阅读位置、章节、朗读历史、段落批注或偏好
- 不上传系统语音选择、收藏音色、语速、字号或亮度设置
- 不读取或上传“个人声音”的原始录音与声音模型
- 不包含广告 SDK、分析 SDK 或云端朗读服务
- 不需要账户、登录或激活密钥

## 本地数据

书籍由用户主动选择并在本机解析。声页只在本机保存“我的图书”文件位置、
观看历史、必要的阅读位置、文字批注、显示偏好和 EPUB 封面缓存，原始
书籍不会被复制到云端。删除观看历史或从“我的图书”移除书籍不会删除原始
文件；从“我的图书”移除书籍时会同时清理对应的本地封面缓存。系统增强或
高级音色由 macOS 下载和管理；声页只读取已经安装的音色。

个人声音的录制、生成、加密存储和授权均由 macOS 管理。声页在用户明确
授权后只能调用系统提供的声音进行实时朗读，不能导出个人声音生成的音频。
