# 声页 1.0.0

声页的首个公开版本现已发布：完全免费、开源、无广告，不需要账户、登录或
激活密钥。所有朗读、分页和阅读进度都在 Mac 本机处理。

## 下载

- `VoicePage-v1.0.0-macOS-universal.dmg`：推荐安装包
- `VoicePage-v1.0.0-macOS-universal.zip`：备用压缩包
- `SHA256SUMS.txt`：安装包完整性校验值

## 本版主要功能

- 导入无 DRM 的 EPUB 2/3 与 TXT
- 单击任意一句话，从当前句开始朗读
- 朗读句子高亮，跨页时自动翻页
- 章节目录、上一章与下一章快捷操作
- 字号变化后自动重新分页，仅需左右翻页
- 系统音色、语速、亮度、日间/夜间/跟随系统外观
- 定时关闭、读完本小节或本章后停止
- 全屏阅读状态栏、键盘和触控板操作、翻页动画
- 自动保存每本书的上次阅读位置

## 系统要求

- 目前只提供 macOS 版本
- macOS 14 Sonoma 或更高版本
- Universal 2，同时支持 Apple 芯片（M 系列）与能够运行 macOS 14 的
  Intel Mac

## 首次打开

当前安装包尚未使用 Apple Developer ID 公证。若 macOS 阻止启动，请在
Finder 中右键“声页”并选择“打开”；如仍被阻止，可前往“系统设置 →
隐私与安全性”选择“仍要打开”。
