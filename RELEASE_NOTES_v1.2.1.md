# 声页 1.2.1｜更完整的本地听读体验

从 v1.0.0 发布以来，我根据实际使用反馈，围绕书籍管理、连续朗读、音色、
批注和阅读控制完成了一轮系统性升级。本次版本重点提高操作效率、朗读连续性
和本地数据管理能力。

声页仍然保持完全免费、开源、无广告。

## 本版更新

### 本地图书管理

- 新增独立封面书架；图书导入后保留在“我的图书”，无需重复选择文件。
- 支持提取并缓存 EPUB 原书封面；TXT 或无封面书籍使用本地占位封面。
- 新增“继续上次阅读”，可从首页直接返回最近保存的阅读位置。
- 观看历史按“今日、昨日、前天、更早”分组，支持单条删除、全选和批量删除。

### 朗读与自动翻页

- 支持单击任意一句话，从当前句开始朗读，并实时高亮正在朗读的句子。
- 新增可开关的“跟随朗读自动翻页”，语音进入下一页时自动定位到对应内容。
- 朗读过程中可自由浏览其他页面，不再被强制带回朗读页；需要时可使用
  “返回朗读位置”恢复跟随。
- 优化自然段和章节之间的连续朗读衔接，减少停顿。

### 本地音色中心

- 增强／高级音色下载、已安装音色管理和个人声音入口统一集中到“音色中心”。
- 支持音色搜索、试听、收藏和品质标记。
- 支持调用经过授权的 macOS“个人声音”；录制、生成和存储仍由系统在本机完成。
- 修复个人声音已经选择但无法正常朗读的问题。

> **音色建议：** 建议下载并使用“黎潋（高音质）”。经中文长文本连续朗读
> 测试，这款声音在语调、停顿和句子衔接方面更加自然。音色是否可用取决于
> macOS 版本、系统语言和当前已下载的语音资源。

### 文字批注

- 批注范围由整段调整为任意文字，可选择一个字符、一句话或一段文字。
- 支持五种高亮颜色、注释、下划线、系统翻译和拷贝。
- 已有注释的右键菜单新增“编辑注释”和“清除注释”。
- 清除注释时保留该文字已有的高亮或下划线。

### 阅读显示与操作

- 修复字号调整可能只触发重新分页、实际字形未及时变化的问题。字号变化后，
  每页文字数量同步调整，全程只需左右翻页。
- 新增白天、黑夜、跟随系统外观、护眼模式和阅读页面亮度调节。
- 完善章节选择、全屏自适应、底部电量／时间／进度状态栏和翻页动画。
- 保留键盘与触控板操作，并在指南中说明“自然滚动”可能造成的方向差异。
- 修复部分 EPUB 目录乱码、单击句子无法朗读和右键菜单出现无关“服务”项等问题。

- `SHA256SUMS.txt`：安装包 SHA-256 完整性校验值

## 支持的设备与系统

- 目前只提供 macOS 版本。
- 最低支持 macOS 14 Sonoma。
- 安装包为 Universal 2，同时支持 Apple 芯片（M 系列）与能够运行
  macOS 14 的 Intel Mac。
- 支持无 DRM 的 EPUB 2/3 和 TXT。
- 系统翻译需要 macOS 14.4 或更高版本。
- “个人声音”需要符合 Apple 要求的 Apple 芯片 Mac、系统语言和授权状态。

数据。朗读、分页、封面提取和阅读记录均在 Mac 本机完成。

## 首次打开

当前安装包尚未使用 Apple Developer ID 公证。如果 macOS 阻止首次启动，
请在 Finder 中右键“声页”并选择“打开”；如果仍被阻止，可以前往
“系统设置 → 隐私与安全性”选择“仍要打开”。
